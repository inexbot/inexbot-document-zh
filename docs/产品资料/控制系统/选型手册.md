# C2200 系列控制器选型手册

## 产品系列概述

纳博特 C2200 系列工业机器人控制器是基于纳博特自研运动控制算法平台开发的通用运动控制器产品线。该系列产品覆盖从标准应用到高端多轴协同的各类工业场景。

## 选型关键参数

选型时应重点考虑以下参数：

| 参数 | 说明 |
| :--- | :--- |
| 控制轴数 | 控制器可控制的最大轴数（含机器人轴 + 外部轴） |
| 通讯接口 | EtherCAT、CANopen、Modbus 等接口数量和类型 |
| IO 数量 | 数字 IO 和模拟 IO 的数量是否满足应用需求 |
| 运动插补方式 | 支持的插补类型（关节、直线、圆弧、曲线等） |
| 工艺包支持 | 焊接、切割、打磨、喷涂等工艺包是否可选配 |
| 扩展能力 | 视觉接口、力控接口等扩展模块支持情况 |

## 典型选型建议

### 6 轴标准焊接应用

**推荐型号**：C2200 标准版

| 配置项 | 推荐配置 |
| :--- | :--- |
| 控制轴数 | 6 轴机器人 + 0~2 外部轴 |
| 通讯接口 | EtherCAT × 1（伺服）+ Modbus TCP |
| IO | 标配 18DI + 18DO |
| 工艺包 | 焊接工艺包（弧焊参数、摆焊、寻位跟踪） |
| 示教器 | T30/T31 竖版焊接示教器 |

**适用场景**：标准弧焊、点焊、薄板焊接

### 6 轴高端切割应用

**推荐型号**：C2200 增强版

| 配置项 | 推荐配置 |
| :--- | :--- |
| 控制轴数 | 6 轴机器人 + 3~5 外部轴（含 Z 轴跟随） |
| 通讯接口 | EtherCAT × 1 + CANopen × 1 + TCP × 2 |
| IO | 标配 + 扩展 IO 模块 |
| 工艺包 | 激光切割工艺包 + DXF 直接导入 |
| 扩展 | 视觉定位接口（可选） |

**适用场景**：激光切割、等离子切割、水刀切割

### 多机协同应用

**推荐型号**：C2200 协同版

| 配置项 | 推荐配置 |
| :--- | :--- |
| 控制轴数 | 支持双机协同（最多 12 轴同步） |
| 通讯接口 | EtherCAT × 2（分别控制两台机器人） |
| IO | 标配 + 安全 IO 模块 |
| 工艺包 | 协同工艺包（碰撞检测、同步控制） |
| 安全 | 协作模式支持 |

**适用场景**：协同搬运、协同装配、人机协作

### 7 轴机器人应用

**推荐型号**：C2200 标准版 + 外部轴扩展

| 配置项 | 推荐配置 |
| :--- | :--- |
| 控制轴数 | 6 轴机器人 + 1 个外部轴（地轨/变位机） |
| 通讯接口 | EtherCAT × 1 + 扩展轴接口 |
| 工艺包 | 对应应用工艺包 |
| 标定 | 外部轴标定功能 |

**适用场景**：大型工件焊接（地轨行走）、多工位协同（变位机）

## 通讯接口选型

| 通讯类型 | 用途 | 是否标配 |
| :--- | :--- | :--- |
| EtherCAT | 伺服驱动通讯（高速实时） | 标配 |
| CANopen | 伺服驱动通讯（中等速度） | 视型号 |
| Modbus TCP | 与 PLC/视觉系统通讯 | 标配 |
| TCP/IP | 通用网络通讯 | 标配 |
| RS485/RS232 | 串口设备通讯 | 标配 |
| OPC-UA | 与 MES/SCADA 系统集成 | 可选配 |
| EtherNet/IP | 与 AB/西门子 PLC 集成 | 可选配 |

## Q&A

**Q：C2200 标准版和增强版有什么区别？**

A：主要区别在于控制轴数和通讯接口数量。标准版最大控制 8 轴，增强版支持最多 16 轴，并提供更多 EtherCAT 和 CANopen 接口，适合复杂多轴协同应用。

**Q：控制器最多能控制多少个轴？**

A：C2200 高端版通过 EtherCAT 总线最多支持 64 轴同步运动控制，可满足大型多机协同系统和复杂自动化产线需求。

**Q：是否需要额外购买工艺包？**

A：基础运动控制功能随控制器标配。焊接、切割、打磨、喷涂等专业工艺包需要额外选配。具体工艺包价格和授权方式请联系纳博特销售。

**Q：控制器支持哪些品牌的伺服？**

A：支持松下（A5/A6）、三菱（MR-J4）、安川（Sigma-7/5）、西门子（SINAMICS V90）、台达（ASDA-A3）、汇川（IS620N）等主流品牌。具体兼容性请参考技术资料中的"支持的伺服型号"文档。

**Q：选型时如何确定是否需要外部轴？**

A：如果应用需要超过 6 个运动轴（如带变位机焊接、地轨行走、多自由度变位机），则需要配置外部轴。6 轴标准焊接、切割等应用使用控制器标配轴数即可。
