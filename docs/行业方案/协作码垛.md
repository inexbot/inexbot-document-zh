# 协作机器人码垛方案

## 产品简介

纳博特协作机器人码垛方案是针对物流、食品、医药等行业开发的完整行业解决方案。纳博特协作机器人搭配专用码垛工艺包，可实现高效、精准的箱体堆叠，适用于多种产品规格和堆叠模式。

纳博特协作机器人以其高安全性、高灵活性和易于部署的特点著称，无需安全围栏即可与人协同工作，大幅降低空间需求和部署成本。

## 产品优势

### 高安全性

协作机器人本身具备碰撞检测功能，在人机协作场景下，如果机器人与操作人员发生碰撞，机器人会立即停止运动并回退，确保操作人员安全。同时，碰撞检测灵敏度可通过参数调节，适配不同负载和场景。

### 灵活部署

协作机器人体积小巧、重量轻，单人即可搬运安装。纳博特专用码垛工艺包提供可视化配置界面，无需编程即可快速适配新产品规格。

### 多种堆叠模式

纳博特码垛工艺支持多种堆叠模式，包括：

- **行列模式**：标准行列排列，支持整体旋转角度（0°、90°、180°、-90°）
- **纵横交错**：相邻层交错堆叠，提高稳定性
- **回字型**：内外圈交替堆叠，适用于特定包装
- **五花垛**：梅花形堆叠，适用于圆形物体
- **自定义**：支持用户自定义堆叠规则

### 便捷的仿真与调试

纳博特提供 PC 端仿真软件，支持离线编程和轨迹仿真。工程师可在 PC 上完成轨迹规划、节拍测算和碰撞检测，大幅减少现场调试时间。

### 高防护等级

协作机器人可选配 IP68 防护等级版本，适用于潮湿、多尘等恶劣环境。
