# 指令运行逻辑

## 试运行逻辑

1. 持续按下使能按钮，程序试运行到一半将光标切换至其他行（不松开使能按钮），程序不会继续往下走。
2. 持续按下使能按钮，程序试运行到一半将光标切换至其他行（松开使能按钮），程序会从光标所在行继续往下走。
3. 试运行只在示教模式下可用。
4. 工艺试运行只走点位，不实际运行工艺，码垛会记录。
5. 机器人在 P001 需要运动到 P002 时，光标会先跳到 P002，机器人再运动到 P002。

## 单步逻辑

1. 持续按下使能按钮，程序单步运行到一半将光标切换至其他行（不松开使能按钮），程序会从光标所在行继续往下走。
2. 持续按下使能按钮，程序单步运行到一半将光标切换至其他行（松开使能按钮），程序会从光标所在行继续往下走。
3. 单步只能在示教模式下使用。
4. 工艺单步只走点位，不实际运行工艺。

## 倒序逻辑

1. 倒序只支持运动指令，未运行的指令会直接跳过。
2. 倒序只支持单步倒序，试运行无法倒序。
3. 倒序只支持示教模式。
4. 机器人在 P002 需要运动到 P001 时，光标会等待机器人运动到 P001，随后再跳转至 P001。

## Q&A

**Q：试运行、单步、倒序分别有哪些使用限制？**

A：
- **试运行**：只支持示教模式；工艺试运行只走点位，不实际运行工艺（码垛除外）
- **单步**：只支持示教模式；工艺单步只走点位，不实际运行工艺
- **倒序**：只支持示教模式；只支持运动指令，未运行的指令会直接跳过；只支持单步倒序，试运行无法倒序

**Q：试运行和倒序时，光标与机器人运动的先后关系是什么？**

A：
- **试运行时**：若机器人当前在 P001、目标为 P002，则光标会先跳到 P002，机器人再运动到 P002
- **倒序时**：若机器人当前在 P002、目标为 P001，则机器人会先运动到 P001，光标再跳转至 P001

**Q：试运行时，在不松开使能按钮的情况下切换光标会怎样？**

A：程序不会继续往下走。需要松开使能按钮后，程序才会从光标所在行继续往下运行。
