# 森峰需求

## LASER_SET指令-激光设置

**功能**：该指令支持在切割过程中实时修改激光加工参数（气压、功率、频率、占空比），修改过程中切割流程无需停顿，可实现动态参数调整。

![](./assets/asenfeng-1.png)

### 参数界面说明

| 参数 | 值 | 注释 |
| :--- | :--- | :--- |
| 工艺号 | 填写所需要使用 的工艺号 | 0-500 |
| 使用临时工艺参数 | 选择使不使用临时参数 | |
| 气压 | 激光切割时需要用到的气压，可绑定变量 | [1-99999]Kpa |
| 激光功率 |激光器每秒钟发出的激光能量，可绑定变量 | [1-99999]W |
| 激光频率 | 激光器每秒钟发出激光的次数，可绑定变量 | [1-99999]Hz |
| 激光占空比 | 激光器输出信号的占空时间比例，可绑定变量| 1-100 |
