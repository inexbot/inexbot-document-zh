# 参数说明

首段焊接：仅焊接首段固定为焊接段，且首段焊接段的长度是固定的，空走距离也设置为固定值

机器人焊接轨迹为：焊接-空走-焊接

注：仅首段焊接距离是固定的
- [参数图示](assets/rgMS_Cr3AIFKn_KpRGFPBrn2wjcbH4sXFIzsg4WSBbA.png)

当焊接轨迹总程度为200时，首段焊接距离设置为80，空走距离也设置为80，那么仅剩的焊接段距离为40
- [如图](assets/FSmuDDNdqJVIRTH_0CGVr4xjfwf9hSDN9ArPZW5WGRA.png)

# 首段焊接

编写程序，填入数据，可以通过设置不同的焊接、空走速度，观察机器人轨迹是否按照焊接-空走-焊接的顺序，且要检查焊接段的长度和空走长度是否和设置的一样。
- [如图](assets/fggEYJIvMLlwPWm3LKvmCybFwIdNWpt7xEtnDMimCK0.png)

# 首段空走

仅焊接首段固定为空走段，且首段空走的长度是固定的，焊接距离也设置为固定值

机器人焊接轨迹为：空走-焊接-空走

- [如图](assets/fqscXhmKoej198z9LyPnK_wuC3whgwB5hi4-IdaeSVY.png)

注：仅首段空走距离是固定的
