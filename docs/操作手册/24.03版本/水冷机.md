# 主要作用

通过纳博特系统设置冷水机的温度，以及获取冷水机当前的温度，流量以及压缩机的转速
- [如图](assets/hG1OK8VF3BDsXxRdBykDXpDuATeWYF9XqOdypV8PTnI.png)
当水冷机使能打开时，可通过底部设置按钮，可对冷水机的温度的进行预设
冷水机的当前温度、流量以及压缩机的转速都是实时刷新的，当退出该界面时，该数据不再刷新
