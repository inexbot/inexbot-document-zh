# 水冷机

## 主要作用

水冷机是激光切割系统的重要辅助设备，用于冷却激光切割头和激光器，防止过热损坏。纳博特控制器支持与水冷机联动控制，实现自动启停和状态监控。

## 主要功能

### 温度设置

通过控制器可设置水冷机的目标温度，系统自动调节制冷功率维持设定温度。

- **目标温度**：激光切割头建议工作温度范围（参考激光器手册）
- **温度偏差报警**：当实际温度超出设定范围时触发报警

### 流量监控

控制器可实时监控水冷机流量和压力参数：

- **冷却液流量**：确保冷却液循环正常
- **压力监控**：检测管路堵塞或泄漏
- **流量报警**：流量低于阈值时触发保护停机

### 联动控制

水冷机与机器人运动联动时：

1. 当激光器开启时，控制器自动启动水冷机冷却
2. 激光器关闭后，水冷机按设定延时关闭
3. 水冷机故障时，控制器自动停止激光输出并报警

## 操作说明

### 启动水冷机

当水冷机控制启用时，可通过底部设置按钮对水冷机的温度进行预设

### 监控状态

水冷机的当前温度、流量以及压降电机的转速都是实时刷新的，当退出该界面时，此数据不再更新

## 参数说明

| 参数 | 值 | 说明 |
| :--- | :--- | :--- |
| 温度设置 | 用户设定 | 目标冷却温度（℃） |
| 流量监控 | 实时显示 | 当前冷却液流量（L/min） |
| 压力监控 | 实时显示 | 当前系统压力（bar） |

## Q&A

**Q：水冷机的主要作用是什么？**

A：水冷机用于冷却激光切割头和激光器，防止过热损坏。激光切割过程中激光器和切割头都会产生大量热量，需要通过冷却系统维持正常工作温度。

**Q：纳博特控制器如何与水冷机联动？**

A：控制器通过 IO 信号或通讯协议与水冷机连接。当激光器启动时，控制器自动启动水冷机；当水冷机故障时，控制器自动停止激光输出并报警，实现联锁保护。

**Q：水冷机温度设置多少合适？**

A：具体温度设置应参考激光器使用手册。常见光纤激光器的推荐工作温度在 20~30℃ 范围内，且应保持温度稳定，避免频繁波动。

**Q：流量报警是什么原因？**

A：流量低于设定阈值通常由以下原因导致：冷却液不足（需要补充）、管路堵塞（需要清洗管路）、水泵故障（需要维修或更换水泵）或过滤器堵塞（需要更换过滤器）。流量报警后应立即停机检查，避免激光器过热损坏。
