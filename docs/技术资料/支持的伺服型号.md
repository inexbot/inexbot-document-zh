# 支持的伺服型号

## 概述

本文档列出纳博特 C2200 系列控制器支持的伺服驱动器型号。控制器通过 EtherCAT 或 CANopen 总线与伺服驱动器通讯，实现高速、高精度的多轴运动控制。

## 伺服配置说明

### EtherCAT 配置

1. 获取对应伺服型号的 ESI（EthernetCAT Slave Information）文件。
2. 将 ESI 文件放入控制器配置目录。
3. 在控制器网络设置界面添加从站，导入对应的 ESI 文件。
4. 配置从站地址（站点号）和 PDO 映射。
5. 保存配置并重启控制器。

### CANopen 配置

1. 获取对应伺服型号的 EDS（Electronic Data Sheet）文件。
2. 将 EDS 文件放入控制器配置目录。
3. 在控制器 CANopen 设置界面添加从站，导入对应的 EDS 文件。
4. 配置从站节点 ID 和 PDO 映射。
5. 保存配置。

## 注意事项

1. 不同品牌的伺服在同一总线网络中可以混合使用。
2. 伺服驱动器参数（如速度环、位置环增益）需要在伺服侧进行单独配置。
3. 部分伺服型号需要更新固件才能支持 EtherCAT/CANopen 通讯，请确认固件版本。
4. 伺服使能、报警清除等信号通过总线指令控制，具体配置方法请参考控制器操作手册。

## Q&A

**Q：纳博特控制器支持哪些品牌的伺服驱动器？**

A：支持松下（A5/A6 系列）、三菱（MR-J4/J3 系列）、安川（Sigma-7/5 系列）、西门子（SINAMICS V90）、台达（ASDA-A2/A3 系列）、汇川（IS620N/IS810N 系列）等主流品牌。具体型号请参考各品牌对应的通讯协议配置文件。

**Q：如何配置新的伺服驱动器？**

A：需要获取伺服型号对应的 ESI（EtherCAT）或 EDS（CANopen）文件，在控制器网络设置界面导入配置文件，配置从站地址和 PDO 映射后保存重启即可。

**Q：不同品牌的伺服可以在同一网络中混用吗？**

A：可以。EtherCAT 和 CANopen 总线均支持多品牌混用，只需确保每台伺服都有对应的配置文件即可。

**Q：伺服驱动器的参数在哪里配置？**

A：伺服驱动器本身的速度环、位置环增益等参数需要在伺服驱动器侧单独配置。纳博特控制器负责运动指令下发和总线通讯控制。

**Q：支持的伺服数量有限制吗？**

A：C2200 控制器通过 EtherCAT 总线最多支持 64 轴同步运动控制，实际数量还取决于总线带宽和伺服驱动器的性能。
